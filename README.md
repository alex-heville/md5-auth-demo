# MD5 Authentication Flaw Demonstration\n\nThis repository demonstrates a common security flaw in using unsalted MD5 for password hashing, and showcases a basic mitigation using salting.\n\n## Files\n\n### naive_auth.py\n\nThis script contains a naive implementation of user authentication using unsalted MD5 hashing.\n\n**Functions:**\n- `register_user(username, password)`: Hashes the password with MD5 and stores the username and hash in `credentials.txt`.\n- `login_user(username, password)`: Hashes the provided password and compares it with the stored hash for the given username.\n\n**Example usage:**\n```python\nimport naive_auth\n\nnaive_auth.register_user(\"alice\", \"password123\")\nnaive_auth.login_user(\"alice\", \"password123\")  # returns True\n```\n\n**Security Issue:** This implementation is vulnerable to pre-computed hash attacks (rainbow tables) because identical passwords produce identical MD5 hashes.\n\n### salted_auth.py\n\nThis script improves upon the naive implementation by adding a random salt to each password before hashing.\n\n**Functions:**\n- `register_user(username, password)`: Generates a cryptographically secure random salt (16 bytes), hashes `salt + password` using MD5, and stores `username:salt:hash` in `credentials_salted.txt`.\n- `login_user(username, password)`: Retrieves the stored salt for the given user, hashes `salt + password`, and compares with the stored hash.\n\n**Example usage:**\n```python\nimport salted_auth\n\nsalted_auth.register_user(\"alice\", \"password123\")\nsalted_auth.login_user(\"alice\", \"password123\")  # returns True\n```\n\n**Security Improvement:** Salting ensures that identical passwords produce different hash values, making preâ€‘computed rainbow tables ineffective. Each user receives a unique salt, which must be stored alongside the hash (salts do not need to be secret).\n\n## Vulnerability Tracking\nA GitHub Issue ([#1](https://github.com/alex-heville/md5-auth-demo/issues/1)) documents the security vulnerability of the naive implementation and proposes the salted solution.\n\n## License\nMIT\n"
